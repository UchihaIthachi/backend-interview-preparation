# Scenario-based Questions

**Scenario/Question:** Database Bottleneck (Oracle) A critical API that fetches user data from Oracle DB has become slow after data growth. How would you diagnose the issue (query vs schema vs indexing)? Would you refactor to NoSQL? Why or why not? How would you optimize without changing business logic?

**Scenario/Question:** NoSQL vs SQL Decision You are building a feature with high read/write throughput and flexible schema. Would you choose Oracle or NoSQL? What trade-offs would you consider? How would data consistency be handled?

**Scenario/Question:** Deadlock Issue Two transactions are causing deadlocks in Oracle. How do you detect and resolve? How do you prevent future occurrences?

**Scenario/Question:** Data Migration You need to migrate data from Oracle to NoSQL. How do you ensure data integrity? How do you validate migration?

**Scenario/Question:** Indexing Strategy A query is slow despite indexing. How do you debug? When can indexes hurt performance?
